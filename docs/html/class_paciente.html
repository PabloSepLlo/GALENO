Detailed Description

Clase Paciente

Esta clase representa a un paciente con sus datos personales, clínicos y sociales, y permite realizar operaciones como cargar datos desde la base de datos, añadir y actualizar, además de manejar relaciones con otras entidades como centro de salud, motivo de ingreso, ayuda social, convivencia y cuidador principal.
